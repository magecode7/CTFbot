# CTFbot
Бот для простого проведения Task-Based соревнований в Telegram.
## Требования
Python 3.10+  
ОЗУ 500мб  
Память 1гб
## Установка
1. Скопируйте проект с GitHub: `git clone https://github.com/magecode7/CTFbot`
2. Установите необходимые пакеты: `pip install -r requirements.txt`
3. Запустите `main.py`
